What is the difference between a static unit like `px` and a responsive unit like `%` or `vh`? 
Px is static and never change its size while % which take up the percentage of the parent. Also vh allows the size of the height to change based on the screen.

Describe what makes a unit responsive and why that matters when building a website.
 
  Building a responsive website matters because depending on the user and their screen you want the website to change its sized based of the screen size. Also while building a responsive website you dont want to be static because it never changes size and  they layouts can break on a certain device like a phone or tablet.
